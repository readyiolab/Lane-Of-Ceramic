# Ceramic Studio Enterprise Backend

Production-grade modular backend scaffold for eCommerce.

## Stack
- Node.js, Express, TypeScript
- MySQL, Prisma
- Redis, BullMQ
- JWT auth + refresh flow
- Cashfree + Shiprocket ready modules
- Swagger/OpenAPI, Docker, PM2, Nginx, GitHub Actions

See `/docs` for architecture, API, deployment, and integrations.
